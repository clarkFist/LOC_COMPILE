## 🔍 版本说明

### 版本号规则

我们采用 [语义化版本控制](https://semver.org/lang/zh-CN/) (SemVer)：

- **主版本号 (MAJOR)**: 当做了不兼容的API修改
- **次版本号 (MINOR)**: 当做了向下兼容的功能性新增  
- **修订号 (PATCH)**: 当做了向下兼容的问题修正

### 发布类型

- 🚀 **正式版本**: 稳定的功能完整版本
- 🧪 **测试版本**: 功能完整但需要更多测试的版本
- 🔬 **开发版本**: 开发中的不稳定版本

### 支持策略

- **最新版本**: 完整技术支持和功能更新
- **前一版本**: 安全更新和重要bug修复
- **更早版本**: 仅提供安全更新
